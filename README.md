

HealthConsult - Frontend UI for Virtual Health Consultations
HealthConsult is a responsive web application designed for virtual health consultations, connecting patients with doctors. 
Technologies Used
React.js: JavaScript library for building the user interface.
Tailwind CSS: Utility-first CSS framework for fast and responsive design.
Vercel: Platform for frontend deployment.
Features
Responsive UI: Fully responsive layout to ensure a seamless experience on mobile, tablet, and desktop.
Available Doctors Page: Display a list of available doctors along with their specialties.
Patient Dashboard: View medical history, upcoming appointments, and messages from doctors.
Book Appointment: Allows patients to book an appointment by selecting a doctor, date, and specialty.
Sign-in Page: Secure login feature with email and password authentication for users.
